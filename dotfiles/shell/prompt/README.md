# Prompt

## Purpose

- Just a convenient prompt for shell using [oh-my-posh](https://github.com/jandedobbeleer/oh-my-posh)

## Usage

- If you use this repo's [zshrc](../zsh/zshrc), you have nothing to do
- If not, use [kema_theme.omp.yaml](./kema_theme.omp.yaml) as config file for [oh-my-posh](https://github.com/jandedobbeleer/oh-my-posh)
